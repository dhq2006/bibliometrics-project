# 文献筛选规则（生成式AI在人文社科与人文计算领域的应用）
## 1. 纳入标准（精准匹配最终692篇文献特征）
### 1.1 研究主题
生成式AI（含AIGC、LLM、GAN、扩散模型等技术）在人文社科、人文计算、数字人文领域的应用、优化、伦理治理等主题，包括但不限于：
- 数字人文领域：古籍智能处理、文化遗产数字化、历史文本分析、文物修复
- 人文社科研究：语言学、社会学、经济学、教育学等学科的AI赋能研究
- 人文计算方法：文本挖掘、语料库构建、计算社会学、文化分析
- 方法与伦理：可解释性、偏见治理、版权合规、学术伦理

### 1.2 文献类型
- Article（原创研究论文）、Review（综述论文）
- 排除会议摘要、新闻、信件、勘误、编辑稿等非学术文献

### 1.3 语言
- 仅English（英文），避免语言数据处理偏差

### 1.4 时间范围
- 2015-2025年（覆盖AIGC技术从萌芽到爆发的完整周期，与最终692篇文献时间分布一致）

### 1.5 数据源
- Web of Science Core Collection（WOS核心合集），保证数据权威性

## 2. 排除标准（明确筛选排除逻辑）
- 非生成式AI主题：仅涉及传统机器学习（如监督/无监督学习）、非生成类AI技术的文献
- 非人文社科场景：AIGC在商业营销、娱乐、普通消费、理工技术研发等非社科领域的应用文献
- 文献类型不符：会议摘要、新闻、信件、书评、勘误等非学术文献
- 语言不符：非英文文献
- 时间不符：2015年前或2025年后的文献（无相关检索结果）
- 数据源不符：非WOS核心合集的文献（如Scopus、CNKI等其他数据库）

## 3. 筛选流程（三阶段闭环，最终导出692篇用于分析）
### 3.1 第一阶段：初筛（主题+数据库限定）
- 操作：基于检索式在WOS核心合集初步检索，得到初始文献；通过数据库限定、时间范围（2015-2025）、语言（English）、文献类型（Article/Review）初步过滤，得到符合基础条件的文献。
- 结果：保留全部符合基础条件的文献进入复筛。

### 3.2 第二阶段：复筛（主题精准匹配）
- 操作：对初筛文献进行标题/摘要筛选，剔除非生成式AI主题、非人文社科与人文计算场景的文献；最终导出692篇用于后续计量分析。
- 结果：导出692篇文献，均符合研究主题要求，且核心字段（作者、标题、年份、DOI）完整。

### 3.3 第三阶段：终筛（质量评价）
- 操作：评估692篇文献来源期刊权威性（优先保留SSCI/A&HCI收录期刊）、研究方法合理性。
- 结果：692篇文献均来自权威期刊，研究方法规范，无低质量文献，全部纳入最终分析数据集。
